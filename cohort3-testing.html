Референс: предыдущие когорты (TFK-124 / TFK-125)

Средние по предметам из двух прошлых волн тестирования TFK-124 / TFK-125 (файл предыдущих результатов). «Тест 2» — полная вторая волна (июль 2025). «Тест 1» — предыдущая волна, доступна только по участникам, сдававшим оба раза, поэтому выборка мала (см. n). Столбец «3-я когорта» — текущий результат для сравнения. Где данных нет — прочерк.
